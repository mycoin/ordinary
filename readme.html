WSCL框架使用手册

一、简介

  • 运行环境

    • IE 6.0/Firefox 2.0/Opera 9.0

  • 框架的组成

    • [WSCL]/wscl.js(提供不同浏览器、编码集和语言的兼容性)、[WSCL]/charset/*.js(不同字符集的处理函数)、[WSCL]/lang/*.js(不同语言的消息提示)、[WSCL]/visual.js(可视化控件库)、[WSCL]/skins/*.js(可视化控制库外观定义)，其中的[WSCL]/表示框架的根目录。

  • 命名规则

    • 在类的属性和对象的属性中，以下划线开头的名称是供内部使用的，如果是变量，第二个字母通常用于标识类型，如_a(Array)、_b(Boolean)、_d(Date)、_e(Element)、_f(Function)、_n(Number)、_o(Object)、_r(RegExp)、_s(String)，请不要对它们进行操作。以$开头的函数用于子类在特定情况下调用，子类需要谨慎使用，不是子类请不要调用这类规则命名的函数。
